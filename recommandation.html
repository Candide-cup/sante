Nos Recommandations

Tilleul

Nom latin : Tilia cordata
Origine : Europe
Parties utilisées : fleurs, bractées

Bienfaits : Calmante, favorise le sommeil et la digestion

Préparation : Infuser 1 c. à soupe dans 250 ml d'eau chaude pendant 10 min

Précautions : Aucune connue

Moringa

Nom latin : Moringa oleifera
Origine : Inde, Afrique
Parties utilisées : feuilles

Bienfaits : Antioxydant, énergétique, riche en vitamines

Préparation : Infuser 1 c. à café dans 200 ml d'eau chaude pendant 5-10 min

Précautions : Éviter pendant la grossesse

Neem

Nom latin : Azadirachta indica
Origine : Inde
Parties utilisées : feuilles

Bienfaits : Purifiant, antibactérien, détoxifiant

Préparation : Infuser 1 c. à café dans 200 ml d'eau chaude pendant 10 min

Précautions : Ne pas consommer en excès

Kinkeliba

Nom latin : Combretum micranthum
Origine : Afrique de l'Ouest
Parties utilisées : feuilles

Bienfaits : Drainant, digestif, tonifiant

Préparation : Faire bouillir 10g dans 1L d'eau pendant 10-15 min

Précautions : Aucune connue

Hibiscus

Nom latin : Hibiscus sabdariffa
Origine : Afrique, Asie
Parties utilisées : calices

Bienfaits : Rafraîchissant, hypotenseur, antioxydant

Préparation : Infuser 1 c. à soupe dans 250 ml d'eau chaude 5-7 min

Précautions : Peut abaisser la tension

Baobab

Nom latin : Adansonia digitata
Origine : Afrique
Parties utilisées : feuilles, pulpe

Bienfaits : Antioxydant, riche en vitamine C, digestion

Préparation : Infuser les feuilles ou diluer la pulpe dans de l'eau

Précautions : Aucune connue

Artemisia annua

Nom latin : Artemisia annua
Origine : Asie
Parties utilisées : feuilles

Bienfaits : Antipaludique, antiviral, antiparasitaire

Préparation : Infuser 1 c. à café dans 250 ml d'eau chaude pendant 10 min

Précautions : Ne pas utiliser pendant la grossesse

Maniguette

Nom latin : Aframomum melegueta
Origine : Afrique de l'Ouest
Parties utilisées : graines

Bienfaits : Digestif, stimulant, aphrodisiaque

Préparation : Infuser ou moudre les graines dans une infusion

Précautions : Éviter en cas d’ulcères gastriques

Aloe vera

Nom latin : Aloe barbadensis
Origine : Afrique du Nord
Parties utilisées : gel, feuilles

Bienfaits : Hydratant, digestif, cicatrisant

Préparation : Utiliser le gel pur ou en jus dilué

Précautions : Peut être laxatif à forte dose

Menthe poivrée

Nom latin : Mentha piperita
Origine : Europe, Asie
Parties utilisées : feuilles

Bienfaits : Antispasmodique, digestif, rafraîchissant

Préparation : Infuser 1 c. à soupe dans 250 ml d'eau chaude pendant 5-10 min

Précautions : Éviter chez les jeunes enfants
